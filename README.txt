This is a simple implementation of TicTacToe programmed in JAVA. 

Instructions:
Compile: javac GameTicTacToe.java
Run: java GameTicTacToe
Commands: Input a number between 1-9 including to play the game. The tiles correspond to the following grid:
1 | 2 | 3
__|___|__
4 | 5 | 6
__|___|__
7 | 8 | 9

Enjoy!
